首先类的定义和c++一样 class 类名{   } 类也有自己的构造函数但是他的构造函数一般不是用他的类名去定义 而是用 function__construct（需要的参数）{ }


有关$this的用法 它是指当前作用的对象


对于类对象的私有化 那么我们在外部是不可以访问到了  但是有一个函数可以帮助我们在类外访问到  就是function __get(私有化的数据){return $this->name}
在类内部定义这样的函数那么我就可以在类外直接访问私有数据  同样我如果是在类外对私有数据的操作 那么就在类内中定义一个functoin __set(私有数据名，改变值){}


对于继承来说PHP中是用extends


当子类中的函数和父类函数重名 子类的函数会将父类的函数覆盖 那么我想在子类访问父类的函数  就需要用到  父类名::函数名（） 一般用于对父类函数的完善


抽象类的关键字用abstract 抽象类中的抽象方法在function前面需要加上abstract 而且抽象方法没有{} 抽象类不能实例化 只能被继承 但是我们如果是想要对派生类进行实例化 就必须要对父类中的全部抽象方法进行函数重载 否则会报错  抽象类的意义是对一部分方法的规范因为继承后不规范就不能对派生类实例化
